# TransFelicianoWeb

Sistema web de gerenciamento de frota desenvolvido para a empresa Trans Feliciano como parte do Projeto de Extensão do curso de Análise e Desenvolvimento de Sistemas do Claretiano – Centro Universitário.

## Sobre o Projeto

O TransFelicianoWeb foi desenvolvido com o objetivo de centralizar e organizar informações relacionadas à gestão operacional da frota da empresa, proporcionando maior controle sobre caminhões, motoristas, rotas, manutenções e despesas operacionais.

A proposta surgiu a partir da identificação da necessidade de modernizar processos que eram realizados de forma manual ou descentralizada, permitindo uma gestão mais eficiente e facilitando o acesso às informações.

## Funcionalidades

* Autenticação de usuários
* Dashboard gerencial
* Gestão de caminhões
* Gestão de motoristas
* Gestão de rotas
* Gestão de manutenções
* Controle de despesas
* Relatórios gerenciais

## Tecnologias Utilizadas

* Node.js
* Express.js
* EJS
* HTML5
* CSS3
* JavaScript
* JSON (armazenamento local de dados)

## Objetivos do Sistema

* Centralizar informações da frota
* Melhorar o controle operacional
* Facilitar consultas e registros
* Reduzir processos manuais
* Apoiar a tomada de decisões
* Melhorar a organização administrativa

## Estrutura do Projeto

O sistema foi desenvolvido seguindo uma arquitetura organizada em módulos, permitindo futuras expansões e melhorias conforme as necessidades da empresa.

## Projeto de Extensão

Este sistema foi desenvolvido como atividade prática de extensão universitária, buscando aplicar conhecimentos adquiridos ao longo do curso em uma situação real de mercado, contribuindo para a melhoria dos processos da empresa parceira.
